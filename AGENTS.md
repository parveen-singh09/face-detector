# Agent Instructions

**Before doing anything else, read [PROGRESS.md](PROGRESS.md).** It's the running
record of how this repo was set up, the current project layout, and what's been
decided. It tells you what's going on.

## Working agreement

- Keep [PROGRESS.md](PROGRESS.md) up to date. When you make a meaningful change
  (scaffold something, add a dependency, install a skill, change structure, make
  a decision), append an entry under the current date. Newest entries at the bottom.
- The Astro app lives in `radiant-ring/`. Run `npm` commands from there.
- Before writing any UI, read `radiant-ring/DESIGN.md` so styling matches the
  Vercel-inspired design system.
